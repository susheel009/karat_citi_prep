Training and Assessment Process

A potential candidate can assess their expertise level by considering their knowledge of the topics listed below and their years of experience. The levels are structured to reflect increasing expertise. For example, to progress to a Senior level, an individual must successfully master the Standard level material.  
After completing the training material for a level, candidates will undergo a Codility test and an interview to evaluate their technical proficiency.

Note: While the material is tailored to meet the specific requirements of Citi Bank, it is also suitable for Java developer upskilling with any client.

Level 1: Standard Java Developer (2 – 3 yrs experience)

Candidates at this level should have a solid understanding of fundamental Java concepts and basic Spring Boot functionalities. They should be able to write and debug simple Java applications and understand basic design patterns. They should also be capable of building REST services independently.

Topics:

Object-Oriented Programming (OOP): Concepts and practical usage.

Inheritance and Composition: Explanation and differences.

Java Fundamentals: Versions 8 and 11, Collections, Exception Handling.

String class: Differences between String, StringBuffer, and StringBuilder; String pool and heap objects.

Iterators: Fail-fast and fail-safe iterators.

Access Modifiers: Explanation and usage.

Types: Differences and usage of types, Classes, Interfaces, Enums, Abstract classes, and Annotations, Anonymous class.

Pass-by-Reference vs Pass-by-Value: Main differences.

Java Versions: New features in Java 8, 11, 17, and 21.

JVM, JRE, and JDK: Differences and functionalities.

Lambda Expressions: Usage and examples.

Collections: Working with ArrayList, Map, Vector, Set; similarities and differences between HashSet and HashMap.

Exception Handling: Exception hierarchy, and custom exceptions, chaining catch blocks.

Debugging: Basic techniques.

HashMap: Storage and retrieval mechanisms.

Thread: ThreadPool Advantages, Creation, and usage. Runnable, Thread class, methods (wait, notify, sleep, start, run, join)

Immutable Classes: Creation and usage.

LinkedList Traversal: Forward and reverse traversal.

Autoboxing/Auto unboxing: Explanation and usage.

Relationship between hash code and equals method: Explanation.

Design Patterns: Understand patterns; Singleton, Factory, Adapter, Observer, Decorator, Bridge, Visitor, Template.

Single-Threaded Application: Implementation.

Spring Framework Fundamentals: MVC, JDBC (running select queries, executing updates, reading result sets), JPA/Hibernate.

Spring Boot Fundamentals: Most used annotations, Application properties, Environment, Auto Configuration.

Spring Framework vs Spring Boot: Differences. Why Spring Boot, Why Containerization, etc.

Beans: Bean lifecycle, Bean definition using XML, Bean definition using Java, types (eager, lazy, prototype scope, etc.).

REST APIs: Writing REST APIs, Defining request and response types, Request validation, API Security basics, REST mappings, HTTP methods, Default HTTP method, HTTP status codes and common errors.

Spring Boot Annotations: @SpringBootApplication and @ComponentScan, @EnableAutoConfiguration, @Controller, @RestController, @Service, @Repository, etc.

Unit Testing Fundamentals: Setup Unit tests in Spring Boot, Basic assertions.

Profiles: Setting active profiles in Spring Boot.

SQL Integration: Calling SQL queries/updates/procedures from Java applications.

Code repository: Bitbucket/Github,

Level 3: Expert Java Developer (5+ yrs experience)

Candidates at this level should have expert knowledge of Java and Spring Boot. They should be able to design and implement highly complex systems, handle performance issues, and have deep understanding of scalability and fault tolerance. They should also be capable of debugging issues and making challenging decisions related to architecture.

Topics:

Design Principles: SOLID, Chain-of-responsibility, KISS, YAGNI, Separation of Concerns, DRY Composition over Inheritance, Inversion of Control, Command Query Separation.

Bitwise Operations: XOR, Left shift, Right shift.

Strings: String formatting for different data types (float, double etc.).

Streams: Find one/all, or else, Throws exception.

HashMap: Internal implementation and thread safety.

IO: IO Streams, Reading/Writing files to/from disk, Serializing/Deserializing objects.

File Conversion: EDI, XML, CSV to JSON.

Immutable Classes and Objects: Creation and usage.

Hash Collision: Handling and reasons for internal changes in HashMap with Java versions.

Spring MVC: Working of Dispatcher Servlet, Regular Expression in Request Mapping.

Spring Boot: Defining custom HTTP error codes, Scheduling, Transactional behavior during errors.

Aspect: Cross-cutting Concerns, Why AOP, Defining Aspects, Pointcut expressions, Joint points, Using AOP to log execution time.

Garbage Collector: Working mechanism.

Performance Issues: Identification and resolution.

Binary Tree: Implementation.

LRU Cache: Implementation.

Threads: Implementing multi-threaded application, Async calls, Synchronized keyword, Critical region.

Custom Connection Pool: Design and creation.

Memory Leaks: Common causes, Prevention, Handling techniques, Cyclic dependencies, JVM GC handling.

Event Driven Messaging Systems: Experience and scale.

Kafka: Scalability, fault tolerance, monitoring.

High Availability Alerts: Common types.

Deadlocks: Detection, tools, and avoidance.

Critical Section of Code: In context of race conditions.

Thread Pool Size: Determination and limitations.

JVM Memory Settings: Configuration.

GC Algorithms: Different algorithms and configurations.

Memory Leak Debugging: Differentiation and resolution.

Latency Issues: Identification and resolution in production systems.
